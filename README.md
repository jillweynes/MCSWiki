## Collaboration guide

1. Clone the repository
1. Install tiddlywiki as a global tool (```npm install -g tiddlywiki```)
1. From the root of the cloned repository, launch the wiki locally (```tiddlywiki . --listen```)
1. Make changes
1. Those changes should automatically update the files on the disk
5. Commit changed files to a new branch, and open a PR
